=> Introduction to HTML

HTML (HyperText Markup Language) is the standard language for creating webpages.
HTML is a markup language, not a programming language.
HTML documents are made of elements, written using tags.







=> HTML Document Structure

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My First Page</title>
</head>
<body>
  <h1>Hello, World!</h1>
  <p>This is my first HTML page.</p>
</body>
</html>


<!DOCTYPE html> → Defines document type (HTML5).
<html> → Root element.
<head> → Metadata section.
<title> → Page title (shown in browser tab).
<body> → Visible content of the page.






=> HTML Tags

Structure of an Element
Opening Tag: <p>
Closing Tag: </p>
Content: "Hello"
Element = Opening + Content + Closing







=> Attributes

Extra info about an element (e.g., class, id).
Syntax: name="value"
Rules:
Space between tag name & attribute.
Multiple attributes separated by spaces.

Empty Elements -> Don’t need closing tag.
Examples: <img>, <br>, <hr>

=> HTML <iframe> Tag

1. Definition

<iframe> = Inline Frame
Used to embed another HTML page, video, map, or other web content inside your current webpage.
It creates a “window” in your page showing another webpage.

2. Basic Syntax

<iframe src="URL"></iframe>
src → The URL of the page or content you want to embed.
