The Anatomy of the Web – How Websites Work?

Client → Your device (laptop, mobile) + browser (Chrome, Firefox).
Server → Special computers storing websites.
Process → Browser sends request → Server responds with a copy of the webpage → Browser displays it.

Introduction to HTML

HTML (HyperText Markup Language) → Standard language for creating webpages.
It is a markup language, not a programming language.
HTML documents are made of elements, written using tags.

HTML Document Structure

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My First Page</title>
</head>
<body>
  <h1>Hello, World!</h1>
  <p>This is my first HTML page.</p>
</body>
</html>

<!DOCTYPE html> → Defines document type (HTML5)
<html> → Root element
<head> → Metadata section
<title> → Page title (browser tab)
<body> → Visible content

HTML Tags

Structure of an Element

Opening Tag: <p>
Closing Tag: </p>
Content: "Hello"
Element = Opening + Content + Closing

Attributes

Provide extra info about an element (e.g., class, id).
Syntax: name="value"

Rules:

Space between tag name & attribute.
Multiple attributes separated by spaces.

Empty Elements (no closing tags): <img>, <br>, <hr>

HTML Texts

Headings & Paragraphs

<h1> → <h6> (biggest to smallest)

Tables in HTML

    <td>History</td>

    <td>N</td>
  </tr>

CSS Borders

table, th, td {

Structure Tags

<thead> → Table header

<tfoot> → Table footer

Attributes

Other Tags

<colgroup> / <col> → Define column styles

Images in HTML

<img src="cat.jpg" alt="Cute Cat">

Required attributes:

src → Path to image
alt → Alternate text

Links in HTML

Basic

<a href="https://www.google.com">Click Here</a>

Attributes

href → Destination URL
target="_blank" → Open in new tab

Bookmarks (within same page)

<a href="#section1">Go to Section 1</a>
<h2 id="section1">Section 1</h2>

Phone & Email

<a href="tel:+01273123456">Call Me</a>
<a href="mailto:someone@example.com">Email Me</a>

Forms in HTML

Basic Example

<form>
  <label for="fname">First name:</label><br>
  <input type="text" id="fname" name="fname"><br>
  <label for="lname">Last name:</label><br>
  <input type="text" id="lname" name="lname">
</form>

Input Types

Text, Password, Email, Number, Tel, URL, Search
Checkbox, Radio, Range, Color
Date, Time, Month, Week, Datetime-local
File, Hidden, Submit, Reset, Button

<label>

Links text with input → improves accessibility

Dropdown (<select>)

<select multiple>
  <option value="volvo">Volvo</option>
  <option value="audi">Audi</option>
</select>

Textarea (Large Input)

<textarea rows="5" cols="30">
Type message here...
</textarea>

action Attribute

Defines what happens after submit:
<form action="/submit_form.php">

HTML <iframe> Tag

Definition

<iframe> = Inline Frame
Embeds another webpage, video, map, etc.
Creates a “window” in your page.

Syntax

<iframe src="URL"></iframe>
src → The URL of the page/content to embed

Semantic Tags in HTML

Non-semantic → <div>, <span>

Semantic → <header>, <footer>, <nav>, <article>, <section>, <main>, <aside>, <figure>, <figcaption>
Benefits → Improves SEO + accessibility
