HTML Notes

1. Headings

HTML provides 6 levels of headings:

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>


Concept:

<h1> → Opening tag
</h1> → Closing tag
Content inside tags → Heading text
Element = Opening tag + Content + Closing tag

2. Paragraphs

<p>This is a paragraph.</p>

3. Line Break & Horizontal Rule

<br>   <!-- Line break -->
<hr>   <!-- Horizontal line -->

4. Text Formatting

<b>Bold Text</b>
<i>Italic Text</i>
<strong>Strong (important) text</strong>
<em>Emphasized (italic) text</em>

Example:

<p>Java <b><i>Full Stack</i></b> - Spring, Spring Boot, Kafka, Docker...</p>

5. Links (Anchor Tag)

<a href="https://www.completejavaclasses.com/testimonial.php">Click Me</a>

Target Attribute:

target="_self" → Default, opens link in same tab
target="_blank" → Opens link in new tab

6. Images

<img src="image.jpg" alt="Photo" width="300" height="200">


src → Image source (URL or path)
alt → Alternative text (shown if image not available)
width & height → Resize image

7. Subscript & Superscript

<p>H<sub>2</sub>O</p>   <!-- Subscript -->
<p>a<sup>2</sup> + b<sup>2</sup></p>   <!-- Superscript -->

8. Tables

<table>
  <tr>
    <th>ID</th>
    <th>Name</th>
  </tr>
  <tr>
    <td>1</td>
    <td>xyz</td>
  </tr>
  <tr>
    <td>2</td>
    <td>abc</td>
  </tr>
</table>


<table> → Defines table
<tr> → Table row
<th> → Table header (bold & centered)
<td> → Table data

Colspan Example:

<td colspan="2">Merged Cell</td>

9. CSS in HTML

Internal CSS → Written inside <style> tag in <head> section.

<style>
  table {
    border: 2px solid red;
    border-collapse: collapse;
  }
</style>




10. Lists in HTML

1. Ordered List (Numbered List)
Concept:

Ordered List is created using <ol> tag.
Each list item is written inside <li> tag.
Numbers (1, 2, 3 …) are shown automatically.

Example:

<h3>Ordered List Example</h3>
<ol>
  <li>Apple</li>
  <li>Banana</li>
  <li>Mango</li>
</ol>


Type Attribute:

type="1" -> Numbers (default)
type="A" -> Uppercase Letters
type="a" -> Lowercase Letters
type="I" -> Upper Roman
type="i" -> Lower Roman

Example with Type:

<ol type="A">
  <li>Math</li>
  <li>Science</li>
  <li>English</li>
</ol>

2. Unordered List (Bulleted List)

Concept:

Unordered List is created using <ul> tag.
Items inside <li> appear with bullets by default.

Example:

<h3>Unordered List Example</h3>
<ul>
  <li>Car</li>
  <li>Bike</li>
  <li>Bus</li>
</ul>


Type Attribute (bullet style):

type="disc" -> Solid circle (default)
type="circle" -> Hollow circle
type="square" -> Solid square

Example with Type:

<ul type="square">
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

3. Nested Lists

Concept:
One list inside another list.
Can combine <ol> and <ul> together.

Example:

<h3>Nested List Example</h3>
<ol>
  <li>Fruits
    <ul>
      <li>Apple</li>
      <li>Mango</li>
    </ul>
  </li>
  <li>Vegetables
    <ul>
      <li>Potato</li>
      <li>Tomato</li>
    </ul>
  </li>
</ol>



11. Forms in HTML

Forms are used to collect user input and send it to a server for processing.
<form> -> Defines a form.
Form elements: <input>, <textarea>, <select>, <label>, <button>, etc.

11.1 Basic Form Structure

<form>
      <label for="itname">Enter Your Name:</label>
      <input
        type="text"
        id="itname"
        name="itname"
        placeholder="Enter your Name"
        required
      />
      <button>Submit</button>
</form>




<form> -> Defines the start of a form.
It is used to collect user input.


<label for="itname">Enter Your Name:</label>
<label> -> Defines a caption or descriptive text for an input field, helping users understand what information they should enter.
for="itname" -> Connects the label to the input whose id="itname".
Clicking the label will focus the input box.
Text shown: “Enter Your Name:”


<input> -> Creates a field where user can enter data.
type="text" -> Specifies it’s a single-line text box.
id="itname" -> Unique identifier for this input (used by <label> or JavaScript).
name="itname" -> The name of the input field; important when sending data to server.
placeholder="Enter your Name" -> Grey hint text shown inside the box until user types something.
required -> User must fill this field; otherwise the form won’t submit.
/> -> Self-closing tag (input has no separate closing tag).


11.2 Input Types

Concept:

The <input> tag supports different types of fields.

Common Input Types:

type="text" -> Single-line text box
type="password" -> Password (hidden text)
type="email" -> Email input
type="number" -> Numeric input
type="date" -> Date picker
type="checkbox" -> Multiple choice options
type="radio" -> Single choice options
type="submit" -> Submit button
type="reset" -> Reset/clear form
